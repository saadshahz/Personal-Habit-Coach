# Release Roadmap

## Phase 1: MVP (Current) - "Insight"
- **Timeline**: Month 1
- **Focus**: Core functionality.
- **Deliverables**:
  - Manual data entry.
  - Basic JSON persistence.
  - Pie chart visualization.
  - Rule-based suggestions.

## Phase 2: "Persistence" - 3 Months
- **Focus**: User accounts and database.
- **Deliverables**:
  - MongoDB integration replacing JSON files.
  - User Authentication (Login/Signup).
  - Multi-device sync.
  - Weekly/Monthly trend reports.

## Phase 3: "Intelligence" - 1 Year
- **Focus**: Advanced AI and Mobile.
- **Deliverables**:
  - React Native Mobile App.
  - LLM integration for highly personalized, context-aware coaching advice.
  - Integration with Calendar/Wearables for auto-tracking.
