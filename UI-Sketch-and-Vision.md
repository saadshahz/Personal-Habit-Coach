# UI Sketch and Vision

## Design Philosophy
**"Clarity through Calmness"**
The UI should feel like a breath of fresh air. We avoid clutter, heavy borders, and dense tables. We use space, soft shadows, and gradients to guide the eye.

## Color Palette
- **Primary**: Deep Navy (`#1e293b`) - Text and strong elements.
- **Accent**: Soft Orange (`#fb923c`) to Vivid Coral - Call to actions / Highlights.
- **Background**: Off-white / Cool Gray (`#f8fafc`).
- **Cards**: White with heavy blur/glassmorphism support.
